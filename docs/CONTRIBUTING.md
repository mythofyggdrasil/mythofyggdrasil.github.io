# Contributing — Myth of Yggdrasil Wiki

Obrigado por querer contribuir! Algumas orientações rápidas para colaborar de forma organizada.

Fluxo recomendado
1. Fork do repositório (se você não for colaborador direto).
2. Crie um branch descritivo:
   git checkout -b feat/mechanics-updates
3. Faça commits pequenos e atômicos.
4. Abra Pull Request para `main` (ou `dev` se criarmos um fluxo com branch de desenvolvimento).

Formato de conteúdo
- Para pequenas correções, edite `index.html`.
- Para conteúdos maiores, crie arquivos em `pages/` (ex.: `pages/equipment.md`) e avise na PR.
- Mantenha português claro e verifique nomes de itens/skills com fontes do jogo.

Issues
- Use o template de bug/feature quando abrir uma issue.
- Assinale se a issue é apenas texto (conteúdo) ou envolve front-end (layout/JS).

Contato
- Marque um membro da staff (por exemplo @ozzi) em PRs que alterem mecânicas do servidor.
