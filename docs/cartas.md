# Sistema de Cartas

Assim como no Ragnarok Online tradicional, as **cartas** desempenham um papel fundamental na personalização e otimização do seu personagem no Myth of Yggdrasil. Elas são a chave para aprimorar seus equipamentos e adaptar sua build para diferentes desafios.

## O que são Cartas?

Para os novos aventureiros, cartas são itens raros dropados por monstros. Cada monstro tem uma chance muito baixa de dropar uma carta que leva seu nome e imagem. Essas cartas podem ser inseridas em equipamentos que possuem **slots** (indicados por `[1]`, `[2]`, etc.) para conceder bônus especiais.

*   **Armas:** Podem ter até 4 slots, geralmente as de nível mais baixo.
*   **Equipamentos (Armadura, Elmo, etc.):** Geralmente possuem 1 slot.
*   **Equipamentos Raros:** Muitas vezes não possuem slots, pois seus atributos base já são muito poderosos.

## Remoção de Cartas

Na área dos ferreiros, você encontrará a NPC **Ophelia**, que tem a habilidade de remover cartas de equipamentos. No entanto, este processo tem consequências:

*   **A carta é destruída no processo.**
*   **O equipamento é preservado.**

Isso permite que você recupere um equipamento valioso para usar uma carta melhor, mas a carta antiga será perdida para sempre.

> **Aviso do Ozzi:** É crucial consultar o **Omnibook (`CTRL + O`)** para verificar o efeito exato de uma carta *antes* de passar horas caçando-a. Os efeitos de algumas cartas podem ter sido alterados no servidor. Acredite, aprendi isso da maneira mais difícil.
